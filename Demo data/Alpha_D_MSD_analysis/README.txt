XML file(s) is the input of this code.
1.xml: SPT analysis demo track data


Below files are outputs of this code.
SPT_demo_1_alpha,
SPT_demo_1_D,
SPT_demo_1_displacement,
SPT_demo_1_msd:  data of all particles in corresponding sample number (ex) 1

per_cell_SPT_demo_alpha,
per_cell_SPT_demo_D,
per_cell_SPT_demo_displacement,
per_cell_SPT_demo_msd: per cell mean data

whole_SPT_demo_alpha,
whole_SPT_demo_D,
whole_SPT_demo_displacement,
whole_SPT_demo_msd: collection of data of all particles of whole input cells (for further processing)
